# Medieval Medical and Apothecary Crafting Suite

The medieval medical/apothecary suite is not the main source of current cultural thinness, but it should still be documented consistently with the second-pass medieval direction.

## Current Surface To Retain

The existing suite can cover common medieval medical and infirmary support:

- Linen bandage rolls
- Apothecary mortars
- Herb pouches
- Field stretchers
- Poultice cloths
- Salve pots
- Apothecary jars
- Cupping horns
- Surgeon tool rolls
- Bone saws
- Splint sets
- Crutches
- Physician bags
- Monastic infirmary kits

## Second-Pass Direction

This suite can remain more culture-neutral than clothing, military, food, and writing. However, future expansions may add regional flavour through:

| Cluster | Optional Additions |
| --- | --- |
| Western/monastic | infirmary chest, herbarium leaf, pilgrim care kit, monastery salve set |
| Islamic urban | apothecary cabinet, syrup jar, distillation prop where supported, scholar-physician kit |
| Byzantine | infirmary lamp, icon-marked healing amulet, hospital ledger prop |
| Rus/Steppe | travel herb pouch, fur-lined medical roll, horseman field bandage kit |
| Song China | scholar-physician box, tea/herb jar, pulse notebook prop, fine ceramic medicine jar |

## Craft Inputs

Use plausible common stocks:

- Linen `Garment Cloth`
- `Spun Yarn`
- prepared leather panels and straps
- pottery clay body and glazed ceramic stock
- metal `Tool Blank Stock`
- herb/remedy stock where current systems support it
- honey, oil, vinegar, salt, or wax stock where appropriate

## Component Boundary

Use live medical components where available. Where runtime medical effects are not implemented, seed props with builder notes explaining their role.

## Test Requirements

Medical tests can remain mostly coverage-oriented, but should verify:

- Bandage-style items use appropriate medical or bandage components.
- Containers and kits use container components.
- Component-gap medical props are documented as props.
- Craft inputs are materially plausible.
