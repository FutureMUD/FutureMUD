---
title: A new home for the FutureMUD engine
summary: futuremud.com is becoming the documentation and release home for the engine.
date: 2026-07-15
tags: website, releases, documentation
---
FutureMUD is moving its engine-facing material away from the LabMUD game website and into a dedicated project home.

The new site is deliberately simple: repository-authored news and pages, downloads promoted by the release workflow, and searchable reference material exported directly from engine metadata. That keeps the public documentation tied to a stable Engine release while leaving LabMUD free to remain its own game.

The first deployment includes reference sections for commands, FutureProg functions and types, collection extensions, and item components. Stable product releases will appear as their product-specific version tags are published.
