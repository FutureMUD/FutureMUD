title: Website and publishing rollout
summary: futuremud.com now hosts product releases, generated engine documentation, and repository-authored project content.
date: 2026-07-16
tags: website, publishing
---
The new FutureMUD project website is now the home for stable product downloads and code-backed documentation.

- Existing public Engine, Database Seeder, Discord Bot, Terrain Planner, and Terrain API releases are available from the downloads page.
- Engine documentation is generated from code metadata and follows the latest stable Engine release.
- Future stable releases use the product tag workflow and resumable website publishing API.
- New product packages are configured as framework-dependent single-file bundles while keeping required settings and content alongside the executable.
- Patch notes can now be published from validated Markdown stored with the website source.
