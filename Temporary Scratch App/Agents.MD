## Purpose of Project
The purpose of this project is a simple console app that lets you run code snippets and get output. It has references to the FutureMUDLibrary and other solution projects, so you can test out code in those contexts without needing to set up a full game or engine environment.

It's not meant for testing, it's usually used for data manipulation, quick prototyping, or running utility code that doesn't fit into the main game projects.
